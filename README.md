# Placement Preparation Tracker

A web-based application developed using **HTML, CSS, and JavaScript** to help students efficiently track their Data Structures and Algorithms (DSA) preparation for internships and placements.

## Overview

Placement Preparation Tracker provides a simple and interactive interface for monitoring DSA preparation progress. The application enables users to organize topics, track solved problems, and visualize their overall preparation status.

The project focuses on building a clean, responsive user interface while strengthening JavaScript fundamentals and DOM manipulation.

---

## Features

- Track DSA preparation progress topic-wise.
- Monitor the number of solved problems.
- Mark completed topics and revision status.
- Interactive and responsive user interface.
- Simple dashboard for visualizing overall progress.
